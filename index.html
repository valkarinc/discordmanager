About the Project

The **Discord Bot Manager** is a powerful desktop application designed to simplify the lifecycle management of your Discord bot projects. Built with **JavaFX**, it provides a streamlined graphical interface for developers to create, import, configure, run, and monitor their bots without wrestling with command-line tools or complex setups.

From a guided bot creation wizard that generates ready-to-code projects (JDA, Discord4J) to real-time console output and dynamic configuration management, this tool empowers you to focus on developing your bot's features, not its infrastructure.

Key Features

✨

Wizard-Driven Bot Creation

Quickly generate new Maven-based Discord bot projects with popular frameworks (JDA, Discord4J) via an intuitive multi-step wizard.

▶️

Effortless Lifecycle Management

Start, stop, and restart individual bots or all running bots with simple, direct controls.

⚙️

Dynamic Configuration

Manage bot settings, environment variables, JVM arguments, and startup delays through an interactive UI.

💻

Real-time Console Output

Monitor live logs from your running bots directly within the application's integrated console.

🌃

Comfortable Dark Mode

Switch to a sleek dark theme for reduced eye strain during extended coding sessions.

📁

Import Existing Projects

Easily add your pre-existing Discord bot projects to the manager for centralized control.
